# Telegram Auto Poster Bot

Guruh va kanallarga avtomatik xabar yuborish tizimi.

## O'rnatish

1. Repositoriyani klonlang
2. `.env.example` ni `.env` ga ko'chiring
3. O'zgaruvchilarni to'ldiring
4. `pip install -r requirements.txt`
5. `python app.py`

## Deploy (Vercel)

1. Vercel ga ulang
2. Environment variables ni sozlang
3. Deploy qiling

## Foydalanish

1. Botda /start bosing
2. Web App ni oching
3. Telefon raqam bilan kiring
4. Guruh va xabarni sozlang
